What is NeuroDot?

A focused dashboard for mental wellness — built to monitor daily focus score, mood patterns, and recommendations, with a friendly UI that avoids noise and keeps signals clear.

  • Clean, blue-toned UI for clarity
  • EEG-inspired wave visualizations
  • Lightweight mood prediction pipeline
  • Accessible, keyboard-navigable controls
